# Introduzione all’editor {#id181OD0QE05Z}

Experience Manager Guides offre un editor di facile utilizzo per la creazione e la modifica di documenti strutturati. L&#39;editor nasconde tutte le complessità della struttura DITA allo scrittore. L&#39;editor fornisce un elenco di elementi DITA che un utente dovrebbe utilizzare all&#39;interno di un documento.

## Nuovo Editor

>[!NOTE]
>
> Il nuovo editor è disponibile in Experience Manager Guides as a Cloud Service a partire dalla versione 2026.05.0. Contatta il team di successo del cliente per abilitare questa funzione.

Il nuovo editor offre un’evoluzione significativa dell’esperienza di authoring, con miglioramenti mirati a livello di prestazioni, usabilità e interfaccia utente/interfaccia utente. Progettato appositamente per le mappe DITA e gli argomenti DITA, consente l&#39;authoring efficiente di contenuti di grandi dimensioni, complessi e altamente strutturati, pur rimanendo completamente compatibile con le versioni precedenti e senza alcun impatto sui contenuti non DITA.

### Funzioni principali

- Miglioramenti delle prestazioni per i file di grandi dimensioni: migliora la reattività per i file DITA di grandi dimensioni con il supporto per gli undo/redo e i marcatori sporchi, garantendo un tracciamento affidabile delle modifiche senza deterioramento delle prestazioni.
- Authoring delle tabelle modernizzato: offre un’esperienza di modifica delle tabelle più intuitiva con riordinamento con trascinamento della selezione, barre degli strumenti contestuali e inserimento di righe o colonne in blocco per una gestione delle tabelle più rapida e accurata.
- Modalità di authoring migliorata: fornisce visibilità configurabile dei commenti e degli attributi XML direttamente in modalità di authoring, riducendo la necessità di passare alla visualizzazione Source quando si lavora con contenuti strutturati.
- Modalità e strumenti di modifica avanzati: introduce le modalità Autore affiancato e Source per gli argomenti e un menu di inserimento rapido, consentendo un authoring strutturato più veloce con un passaggio di contesto minimo.
- Perfezionamenti di interfaccia utente e interfaccia utente: migliora la leggibilità e il comfort visivo tramite un tema scuro e CSS predefinito più ricco nelle modalità Creazione e Anteprima, offrendo un’interfaccia di modifica più moderna.
- Impostazioni editor centralizzato: offre un pannello di impostazioni unificato per personalizzare l’esperienza di authoring, che include il controllo su tag, attributi, commenti XML, spazi unificatori e menu di inserimento rapido nell’editor.

### Limitazioni del nuovo editor

Il nuovo editor non supporta attualmente le seguenti funzionalità. Queste funzionalità saranno introdotte nelle prossime versioni.

- Authoring con la funzione Revisioni abilitata (tutte le revisioni esistenti vengono mantenute e trattate come contenuto normale)
- Ridimensionamento immagini nella vista Autore


>[!NOTE]
>
>La vista Layout per le mappe DITA non sarà supportata con il nuovo editor.


## Editor precedente

Il vecchio editor di Experience Manager Guides offre un’esperienza di authoring semplice e affidabile per i contenuti strutturati, astrazione delle complessità di DITA e garanzia di conformità con gli standard supportati come DITA 1.2, 1.3, DITA leggero e DITA specializzato. Fornisce strumenti essenziali per lavorare con elementi in linea e blocchi comuni, rendendolo adatto per la creazione di contenuti coerente e controllata all’interno di un framework strutturato.
