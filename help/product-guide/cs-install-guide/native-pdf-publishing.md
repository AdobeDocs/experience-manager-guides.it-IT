# Pubblicazione PDF nativa

Con AEM Guides, puoi generare PDF di singoli argomenti o un file di mappa utilizzando la funzione di generazione dell’output di PDF nativo. Puoi personalizzare e utilizzare i modelli di PDF durante la generazione dell’output di PDF. Puoi creare e progettare layout di pagina per presentare le varie sezioni nell’output di PDF in base alle tue esigenze.

Inoltre, AEM Guides supporta anche le variabili di lingua per aiutarti a definire le stringhe localizzate. Consente inoltre di definire stili per vari elementi presenti nell’output di PDF. Se sei un utente avanzato, puoi anche lavorare direttamente con i CSS per creare layout di pagina avanzati.
