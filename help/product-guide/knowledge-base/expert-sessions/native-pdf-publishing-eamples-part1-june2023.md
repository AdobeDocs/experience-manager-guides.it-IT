# Pubblicazione nativa PDF - Apprendimento per esempi - 01

Con il motore di pubblicazione nativo di PDF, è possibile controllare il layout del contenuto del documento manipolando le sezioni desiderate in base alle esigenze utilizzando CSS, HTML e JavaScript facili da aggiornare

## Panoramica della sessione

In questa sessione, con l’aiuto di alcuni esempi scoprirai come utilizzare le funzioni di pubblicazione di PDF nativi .

## Elementi principali da ricordare

In questa sessione hai appreso le seguenti informazioni per la pubblicazione nativa su PDF:
- Abilitazione di funzioni di base come MathML, filigrana e revisioni
- Utilizzo di font personalizzati
- Utilizzo dei metadati da XML DITA nell&#39;output di PDF
- Creazione di un output PDF a più colonne e modifica delle dimensioni della pagina nell’output di PDF (per la stampa)
- Modifica dell’ordine di rendering di alcuni elementi (tramite JavaScript)
